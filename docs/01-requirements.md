# ① 要件定義書: 借金返済バトルRPG「ザワ...ザワ...」

## 1. プロダクト概要

**コンセプト**: 借金返済をRPGのボス戦に見立て、返済をゲーム化する自動管理アプリ

**ターゲットユーザー**: 家計簿が続かない・意思力に頼れない借金返済者（= あなた）

**最重要原則**:
- 「記録しなくても回る仕組み」を最優先
- ユーザーの手動入力はゼロに近づける
- 借金返済をゲーム感覚で続けられること

## 2. ユーザープロフィール

| 項目 | 値 |
|------|------|
| 月収（手取り） | 32万円 |
| 固定費 | 6万円 |
| 借金総額 | 約400万円 |
| 借金種類 | 奨学金・クレカ・ローン・消費者金融 |
| 完済目標 | 2年以内（24ヶ月） |
| 月あたり返済必要額 | 約18〜20万円（利息込概算） |
| 生活費（変動費） | 約6〜8万円/月 |

## 3. 機能要件

### 3.1 ゲームシステム（コア機能）

#### ボスバトルシステム
- 各借金 = 1体のボスモンスター
- ボスのHP = 借金残高
- 返済 = ボスへの攻撃（ダメージ = 返済額）
- ボス撃破 = 完済

| ボス名 | モデル | 特徴 |
|--------|--------|------|
| 奨学金の亡霊 | 奨学金 | 低攻撃力（低金利）・高HP |
| リボの悪魔 | クレカ | 毎月HP回復（利息）・中HP |
| 金利の竜 | ローン | 高攻撃力（高金利）・高HP |
| サラ金の魔王 | 消費者金融 | 最高攻撃力・最高利息 |

#### レベル・経験値システム

| 行動 | 獲得XP |
|------|--------|
| デイリーログイン | 10 XP |
| 今日の予算内で過ごす | 50 XP |
| 最低返済実行 | 100 XP |
| 繰上返済（1万円あたり） | 200 XP |
| ボス撃破（完済） | 1,000 XP |
| 月間予算達成 | 500 XP |

#### 称号システム

| レベル | 称号 |
|--------|------|
| 1 | 借金奴隷 |
| 5 | 見習い戦士 |
| 10 | 借金ハンター |
| 20 | 節約マスター |
| 30 | 財務の達人 |
| 50 | 完済王 |

### 3.2 ダッシュボード

毎日見たい情報:
- 借金残高（総額 + ボス別）
- 前日比（昨日からの変動）
- 今月返済額
- 今日使える金額
- 今月残予算
- 完済予定日
- 現在レベル・XP・次のレベルまで
- ボスHP残量（プログレスバー）

### 3.3 自動化機能

- 借金残高の自動計算（利息含む）
- 返済スケジュール自動管理
- 日次予算の自動算出
- LINE通知（毎朝のステータス報告）
- 月初の自動リセット（月予算再計算）

## 4. 非機能要件

- PWA対応（iPhoneホーム画面に追加可能）
- レスポンシブ（スマホファースト）
- オフライン表示対応（最終同期データ表示）
- 初期設定は10分以内に完了
- 日次の手動操作: ゼロ（通知を見るだけ）
- 月次の手動操作: 借金残高の確認・更新（5分以内）
